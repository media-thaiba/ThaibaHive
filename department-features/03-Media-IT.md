# ThaibaHive — Features for Media & IT Department

Dear Media & IT Team,

ThaibaHive is being built as the unified digital workplace for all Thaiba Garden institutions. Below are the features planned that are relevant to your department. Please review and let us know if anything needs to be added, modified, or removed.

## Features

1. **Help Desk / IT Support** — Ticket creation, assignment, priority levels, comment threads; SLA tracking for issue resolution
2. **Asset Management (IT Assets)** — QR-tagged equipment tracking; purchase/warranty/service history; check-in/check-out
3. **Announcements** — Create and manage announcements with read receipts and priority levels
4. **Events Calendar** — Create institution/department events with RSVP tracking
5. **Circulars & Document Repository** — Upload and manage digital documents by category/department
6. **Staff Polls & Surveys** — Create polls targeting specific roles, departments, or institutions
7. **Staff Directory** — Searchable directory for all staff with contact info and roles
8. **Attendance Management** — Staff attendance with NFC tag / QR code / manual check-in; late/early detection
9. **Approval Center** — Review and approve IT-related requests (help desk escalation, asset requests)
10. **Task Management** — IT project and maintenance task tracking with Kanban board
11. **Daily Reports** — End-of-day reporting for IT activities
12. **Staff Recognition** — Kudos and recognition posts
13. **Grievances** — Submit and manage IT-related or workplace feedback
14. **Audit Log** — Full action trail for IT system changes and access
15. **Diagnostics & Telemetry** — Client-side diagnostic logging with bug report button
16. **Export** — Data export for IT reports
17. **Staff Availability** — Live presence status for team coordination
18. **Purchase Requests** — IT equipment/software purchase requests with multi-stage approval

## We Need Your Input

1. Are there any features listed above that should be **removed or modified** for the Media & IT Department?
2. Are there any **additional features** you need that are not listed (e.g., network monitoring, software license management)?
3. Are there any **overall app features** you believe should be included for all departments?

# ThaibaHive — Features for PR Department

Dear PR Department Team,

ThaibaHive is being built as the unified digital workplace for all Thaiba Garden institutions. Below are the features planned that are relevant to your department. Please review and let us know if anything needs to be added, modified, or removed.

## Features

1. **Announcements & Notices** — Create and broadcast announcements to all institutions, specific departments, or roles; read receipts; priority levels
2. **Events Calendar** — Publish institution events with RSVP tracking and attendee management
3. **Circulars & Document Repository** — Store press releases, media kits, brochures, and official communications; searchable by category
4. **Staff Polls & Surveys** — Gather staff opinions on PR initiatives, events, and institutional matters
5. **Staff Recognition** — Post kudos, Employee of the Month, work anniversaries, and birthday wishes
6. **Grievance / Suggestion Box** — Manage public-facing feedback; respond to named and anonymous submissions
7. **Visitor Management** — VIP visitor check-in/out with host notification and visitor pass
8. **Staff Directory** — Find and contact staff for PR-related coordination
9. **Event RSVPs** — Monitor attendance confirmations for events
10. **Attendance** — Track PR team attendance and availability
11. **Task Management** — Manage PR campaign tasks and deadlines
12. **Daily Reports** — End-of-day PR activity summaries
13. **Approval Center** — Approve/reject communications and event proposals
14. **Expense Claims** — Submit PR event expenses and promotional material costs
15. **Purchase Requests** — Request promotional materials, banners, printing services
16. **Staff Timeline** — Chronological view of PR team activity

## We Need Your Input

1. Are there any features listed above that should be **removed or modified** for the PR Department?
2. Are there any **additional features** you need that are not listed (e.g., social media integration, press release templates, media contact database)?
3. Are there any **overall app features** you believe should be included for all departments?

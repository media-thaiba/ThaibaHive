# ThaibaHive — Features for Project Department

Dear Project Department Team,

ThaibaHive is being built as the unified digital workplace for all Thaiba Garden institutions. Below are the features planned that are relevant to your department. Please review and let us know if anything needs to be added, modified, or removed.

## Features

1. **Task Management (Kanban)** — Full project task board with To Do, In Progress, Review, Completed columns; priorities, due dates, comments, file attachments
2. **Daily Reports** — End-of-day task-based reporting with hours logged; submission compliance tracking
3. **Attendance Management** — Staff check-in/out via NFC, QR, or manual; shift-based with grace periods
4. **Approval Center** — Unified inbox for project-related approvals
5. **Purchase Requests** — Multi-stage approval (Staff → HOD → Accounts → Purchase); audit trail
6. **Expense Claims** — Bill submission with photo upload; approval workflow; reimbursement tracking
7. **Asset Management** — Track project equipment and materials; QR-tagged check-in/out
8. **Staff Directory** — Find and contact project staff across campuses
9. **Staff Availability** — Live presence status (Available, Busy, On Site, On Leave)
10. **Announcements** — Project-wide announcements and updates
11. **Events Calendar** — Project milestones, meetings, and deadlines
12. **Help Desk** — Report project site issues or IT needs
13. **Circulars** — Store and share project documents, blueprints, and plans
14. **Staff Timeline** — Chronological view of each team member's activity
15. **Performance Reviews** — Quarterly project staff evaluations

## We Need Your Input

1. Are there any features listed above that should be **removed or modified** for the Project Department?
2. Are there any **additional features** you need that are not listed (e.g., project Gantt charts, resource allocation, contractor management)?
3. Are there any **overall app features** you believe should be included for all departments?

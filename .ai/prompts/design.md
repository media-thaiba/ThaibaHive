# Premium Design Prompts Template

Copy and execute these prompt templates when redesigning components or adding animations.

---

## Model Prompts

### 1. Refactoring Bland Table Layout into Premium Surface Card Layout
```markdown
Redesign the [ViewName] table view layout to be premium and modern.
Requirements:
1. Apply subtle background gradients and border lines rather than solid borders.
2. Animate items entry using Framer Motion with standard staggered child variants.
3. Replace hardcoded status colours with appropriate shadcn Badges.
4. Integrate search field debouncing and dynamic loaders.
```

### 2. Micro-Interactions Hover Transitions
```markdown
Add dynamic Framer Motion hover and click animations to [WidgetComponent].
Requirements:
1. Ensure hover scale does not overflow layout margins.
2. Respect reduced-motion preferences.
```

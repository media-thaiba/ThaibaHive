# ADR-005: Android Glance Home Screen Widgets

* **Status**: Accepted
* **Date**: 2026-07-18

## Context
ThaibaHive mobile app users require home screen widgets to quickly check tasks, checklists, and schedules.

## Problem
Developing native widgets in Java/Kotlin using raw AppWidgetProvider is error-prone, hard to keep responsive, and difficult to connect with Flutter state.

## Options
1. Raw Android AppWidgetProvider
2. Jetpack Glance with local Room cache and WorkManager sync

## Decision
Choose Option 2: Android Jetpack Glance.

## Consequences
- Widgets are built using a declarative Kotlin layout system.
- To prevent credential exposure and battery drain, widgets do not fetch from network; they read exclusively from a local Room database cache pre-fetched by the companion app.
- WorkManager periodic sync is set to a minimum of 15 minutes.
- Widgets display an amber alert badge if data is older than 30 minutes.

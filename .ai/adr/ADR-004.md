# ADR-004: Zustand Client State Management

* **Status**: Accepted
* **Date**: 2026-07-14

## Context
Client-side UI elements require shared state triggers (e.g. command palette toggle, active sidebar toggles).

## Problem
Using standard React Context triggers expensive page re-renders across all child components.

## Options
1. Redux Tool Kit (RTK)
2. React Context
3. Zustand

## Decision
Choose Option 3: Zustand.

## Consequences
- Clean, hook-based state subscriptions.
- Minimizes boilerplate and prevents redundant re-render actions.

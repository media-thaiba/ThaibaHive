# ADR-003: Drizzle ORM Selection

* **Status**: Accepted
* **Date**: 2026-07-14

## Context
A type-safe database query builder is required to map relationships and execute migrations across SQLite and PostgreSQL.

## Problem
Prisma is heavy, introduces schema translation layers, and has poor support for SQLite-PostgreSQL dialector switching under a single schema definition.

## Options
1. Prisma
2. TypeORM / Typesafe SQL query builders
3. Drizzle ORM

## Decision
Choose Option 3: Drizzle ORM.

## Consequences
- Enables pure TypeScript schemas and type inferences.
- Lightweight operations with zero translation overhead.
- Direct mapping support for both SQLite dev.db and production PostgreSQL dialects.

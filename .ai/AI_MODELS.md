# AI Models Capability Matrix

Guidance on which model to select for specific development and architectural tasks.

| Model / Assistant | Best At | Avoid |
|---|---|---|
| **Antigravity 2.0 (Gemini 3.5)** | Pair programming, orchestrating workflow steps, comprehensive system audits, creating artifacts. | Complex file merges without verification step. |
| **Claude Code** | Long coding sessions, complex logic integration, writing E2E test suites, complex refactoring. | UI visual polish adjustments without layout guidelines. |
| **ChatGPT (GPT-4o / o1 / o3-mini)** | Structural design patterns, high-level planning, debugging complex stack exceptions, writing initial PRDs. | Massive code modifications without exact context files. |
| **Qwen (2.5 Coder)** | Fast backend routes, writing database transactions, writing migrations scripts, single-function logic. | High-level product strategies and complex UI flow alignments. |
| **Cursor / Copilot** | Fast edits, inline comment blocks, quick function refactoring, code auto-completions. | Strategic changes, multi-file architectural updates. |
